At **Neuroinformatics 2015** I described the framework supporting **NCANDA**, a
multi-site longitudinal study of 808 adolescents across five U.S. sites. The system
integrates heterogeneous instruments for clinical, neuropsychological, and MRI data
at baseline and follow-up visits, built on established tools like REDCap and a
lightweight data pipeline.

A key theme: reusing existing software infrastructure to build scalable
neuroinformatics platforms, and using version-control systems as an innovative way
to capture data asynchronously across geographically distributed sites while
maintaining quality control.
