# COMPASS

COMPASS is the platform I helped build at Maze Therapeutics for target
discovery and validation. It combined a data lake and knowledge graph over
clinico-genomic datasets with analytics interfaces and a data-science
environment for computational biology — built on an AWS cloud architecture with
GitOps delivery (Terraform, Kubernetes/Helm, ArgoCD).
