# GREX

GREX is a system for recording, tracking, storing, finding, and retrieving
computational results — capturing provenance and metadata so that derived data
from precision-medicine research is discoverable and reusable. See the talk,
*Meaningful Metadata at Scale: Removing Barriers to Precision Medicine
Research*.
